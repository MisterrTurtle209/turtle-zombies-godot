# Turtle Zombies - Godot 4 Port

## How to Run

1. Download Godot 4.3 or later from https://godotengine.org/download
2. Open Godot
3. Click "Import" and select the `project.godot` file in this folder
4. Click "Run" (F5) to play

## Controls

- **WASD** - Move
- **Mouse** - Look
- **Left Click** - Shoot
- **R** - Reload
- **Shift** - Sprint
- **C** - Crouch
- **ESC** - Pause

## Current Features (Ported from Three.js)

✅ FPS movement with sprint/crouch/stamina
✅ M1911 pistol with shooting, recoil, reload, ammo system
✅ View bobbing + sprint FOV increase
✅ Breakable window barricade (6 planks + sandbags)
✅ Pause menu with resume/restart
✅ Health/points/ammo HUD
✅ Clean square room map
